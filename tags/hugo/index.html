用 Travis CI 輕鬆部署 Hugo Blog 到 Github Page

@kuotinyen · Feb 22, 2019 · 3 min read

前陣子使用 Medium 記錄學習的一些心得感覺很不錯， 但是不支援 code highlight & markdown 對工程師來說真的有點不方便。 嘗試過 Hexo，看了一些 Jekyll 的資料，還曾考慮用 github 的 issue 來寫 Blog， 最後用 Hugo 覺得很適合就決定是它了。 折騰了幾天，參考一些網路上的資源順利完成了。 現在只管生出 markdown 文章， commit 到 github.io 後， Travis CI 就會幫我自動將文章部署到 Github Page 上，太爽了。
